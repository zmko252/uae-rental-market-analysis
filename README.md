# UAE Rental Market Analysis & Rental Price Prediction

## Project Overview

This project analyzes over 73,000 UAE property listings to identify rental price trends, supply and demand patterns, and develop machine learning models for rental price prediction.

## Business Objectives

- Analyze rental price trends across UAE cities
- Identify factors affecting rental prices
- Compare supply and demand across locations
- Build rental price prediction models
- Generate recommendations for stakeholders

## Dataset

- Source: Kaggle
- Original Data Provider: Bayut.com
- Size: 73,000+ property listings
- Coverage: UAE

## Tools Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Seaborn
- Matplotlib
- Folium

## Machine Learning Models

- Linear Regression
- Random Forest
- Gradient Boosting
- Bagging

## Key Findings

- Location is the strongest determinant of rental prices.
- Dubai and Abu Dhabi have the highest rental prices.
- Random Forest achieved the highest predictive accuracy.


## Recommendations

- Implement adaptive pricing strategies.
- Focus investments on high-demand locations.
- Improve prediction models with economic indicators.
